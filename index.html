About SconSolver

SconSolver is a constraint solver written in Scala. It is still under havey development but can be used to solve some simple problems.

Example

The classical send-more-money problem implemented with SconSolver: 

val sendMoreMoneyCSP = new BasicCSPModel {
  val vars = List.fill(8)(newIntVar(0 to 9))
  val List(s, e, n, d, m, o, r, y) = vars
  distinct(vars)
  satisfy((s !== c(0)) && (m !== c(0)))
  satisfy((s * c(1000) + e * c(100) + n * c(10) + d +
           m * c(1000) + o * c(100) + r * c(10) + e) ===
          (m * c(10000) + o * c(1000) + n * c(100) + e * c(10) + y))
  override def solutionStoreToSolution(s: Store) = {
    List(s, e, n, d).map(s(_).value).mkString("") + " + " +
    List(m, o, r, e).map(s(_).value).mkString("") + " = " +
    List(m, o, n, e, y).map(s(_).value).mkString("")
  }
}
